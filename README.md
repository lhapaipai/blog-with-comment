# Blog with Comments (Next.js 14)

réécriture du template : https://vercel.com/templates/next.js/blog-with-comments avec App Router

## Installation

Installer en local Redis.

```bash
pnpm install
pnpm dev
```

## Technos

- Next.js (app)
- Redis (pas convaincu dans cet exemple)
- TailwindCSS
